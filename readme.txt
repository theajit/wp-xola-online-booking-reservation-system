== Description ==

Xola.com is a unified booking, marketing, and distribution system that powers businesses offering tours, activities, classes and experiences.  Xola software helps businesses manage their back-office and online reservations, payment processing, calendaring, inventory, employees, and customers.

Xola is the #1 rated booking system for tour guides and activity providers.  The Xola WordPress Plugin makes installing online booking on your website easy and fast.

== Installation ==	

1. Free sign-up - Register for [Xola.com](https://xola.com/overview)
2. Install Xola E-Commerce WordPress plugin
3. Drop in Xola book now buttons in just a couple clicks!

Once you have your Xola account set up with all your offerings, installing the WordPress plugin takes 30 seconds max and all your offerings will be imported to WordPress for easy activation.
